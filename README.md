# Modular Machine Learning Dashboard

A comprehensive, interactive multi-page Streamlit dashboard designed to streamline the machine learning workflow from data ingestion to evaluation.

## 🚀 Features
* **File Uploading:** Support for CSV and Excel files with live data previews and description tables.
* **Data Preprocessing:** Imputation of missing values, outlier detection/handling, power transformations, PCA, RFE, and class imbalance handling (SMOTE).
* **Visualization:** Custom interactive trends using Line, Scatter, and Box plots via Seaborn.
* **Model Training:** Train Classification algorithms (RF, LogReg, SVM, etc.), Linear Regression, or K-Means Clustering.
* **Evaluation:** Interactive generation of Confusion Matrices, Classification reports, R² analysis, residual plots, and silhouette analysis.
